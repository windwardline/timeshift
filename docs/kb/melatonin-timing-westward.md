# Melatonin for Westward Trips

Why westward travel usually needs less melatonin, and how to use it if you do.

## Often unnecessary

Westward travel requires a phase delay — a later clock — which aligns with the body's natural
tendency to run long. Because the easier direction often resolves with light and sleep timing
alone, many travelers skip melatonin entirely going west.

## If you choose to use it

When it is used westward, timing is oriented to the later destination bedtime rather than an
earlier one. The aim is still to reinforce the local night signal, just shifted later than an
eastward plan would call for.

## Lean on light instead

For westward trips the strongest lever is usually evening light to delay the clock, plus
staying up to the local bedtime. Consider melatonin a minor, optional addition here rather
than the centerpiece it can be for eastward travel.
